## Navigation Guides

### Shopify Admin Quick Reference
- Settings: Admin > Settings (bottom left gear icon)
- Apps: Admin > Apps (left sidebar)
- Products: Admin > Products
- Orders: Admin > Orders
- Analytics: Admin > Analytics
- Themes: Admin > Online Store > Themes
- Domains: Admin > Settings > Domains
- Payments: Admin > Settings > Payments
- Checkout: Admin > Settings > Checkout

### Google Cloud Console Quick Reference
- Projects: console.cloud.google.com > top dropdown
- IAM & Permissions: Navigation menu > IAM & Admin
- APIs: Navigation menu > APIs & Services > Library
- Billing: Navigation menu > Billing
- Storage: Navigation menu > Cloud Storage
- Logs: Navigation menu > Logging > Logs Explorer
