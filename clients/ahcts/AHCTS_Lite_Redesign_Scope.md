# AHCTS — Lite-Scope Website Redesign
**Client:** Aim High Consulting Tutoring Services (ahcts.ca)
**Prepared by:** Makepeace Web Design
**Date:** March 13, 2026
**Engagement Type:** Lite Scope — Restructure & Refresh (WordPress)
**Rate:** $32/hr | Cap: 3 hrs/week

---

## 1. Project Overview

AHCTS serves parents of children with special needs — including Autism, ADHD, Dyslexia, Learning Disabilities, and Behavioral Disorders — and needs its website to reflect that with clarity, warmth, and trust. This engagement is a focused, lite-scope refresh of the existing WordPress site: no new pages, no theme overhaul, no heavy development. The goal is to make the site more emotionally resonant, easier to navigate, and more likely to convert a visiting parent into a booked consultation.

---

## 2. Scope of Work

### 2.1 Homepage

- Rewrite the above-the-fold headline to speak directly to a parent's pain point (e.g., "Your child deserves support that actually works.")
- Add one clear, prominent call-to-action above the fold — linked to the contact or booking page
- Restructure existing services content to be organized by condition (Autism, ADHD, Dyslexia, Learning Disabilities, Behavioral Disorders) in plain, parent-friendly language
- Surface existing testimonials in a more prominent position on the page
- Review and improve whitespace, button contrast, and overall visual warmth within the existing theme — no theme replacement

### 2.2 Navigation

- Simplify the navigation structure using existing menu items — remove redundancy, improve label clarity
- Ensure contact information is visible throughout (header or footer, consistently)

### 2.3 Mobile Responsiveness

- Review the site on mobile and tablet viewports
- Identify and fix any obvious layout breaks, overlapping text, or unreadable sections within the existing theme's capabilities

### 2.4 Accessibility (WCAG AA — Baseline Check)

- Audit color contrast on primary text, headings, and CTA buttons (WCAG AA minimum: 4.5:1 for body text)
- Confirm minimum font size is no smaller than 16px for body copy
- Confirm tap/click targets are adequately sized for mobile (minimum 44x44px recommended)
- Flag any issues found; implement fixes where possible within the existing theme

### 2.5 Basic SEO Cleanup

- Review and rewrite page titles and meta descriptions for the homepage and key pages (using existing SEO plugin — no new plugins)
- Review heading structure (H1, H2, H3) and correct hierarchy where broken
- Confirm image alt text is present and descriptive on key visuals

---

## 3. Out of Scope

The following are explicitly outside this engagement and will not be addressed:

- New pages of any kind
- New plugins or tools not already installed on the site
- Full theme replacement or major visual redesign
- Custom development or code-level functionality changes
- E-commerce or payment integration
- Blog strategy or content creation beyond inline copy edits
- Google Ads, social media, or any off-site marketing
- Hosting, domain, or WordPress core/plugin update management (unless separately agreed)

---

## 4. Notes and Assumptions

- All work is performed within the existing WordPress theme and plugin stack. If a desired change is not achievable without a new plugin or theme modification, it will be flagged and discussed before any additional work is undertaken.
